# 本地文件知识库 MVP 开发计划

## 范围

第一版做桌面本地工具主链路：

1. 多级分类树。
2. 文件导入记录。
3. 标签、备注和分类编辑。
4. 关键词搜索。
5. 规则归档推荐。
6. 文件打开和所在文件夹打开。

明确放到第二版：

1. 本地 Embedding。
2. PDF、Word、Excel 全文解析。
3. 语义搜索。
4. OCR。
5. SQLite 和向量库。

## 技术选择

当前目录为空，为了快速形成可运行 MVP：

- Electron 桌面壳。
- 原生 HTML/CSS/JavaScript。
- JSON 文件做本地数据存储。
- Node/Electron IPC 负责文件选择、打开和数据持久化。

后续可以把 JSON 存储平滑替换成 SQLite。

## 验收标准

1. 应用启动后展示三栏工作台。
2. 可以新增、编辑、删除多级分类。
3. 可以选择本地文件导入记录。
4. 导入时根据规则推荐分类和标签。
5. 可以按关键词搜索文件名、标签、备注、分类。
6. 可以编辑文件分类、标签、备注。
7. 可以打开文件或打开所在文件夹。
8. `npm run build` 能完成基础语法和文件存在检查。
